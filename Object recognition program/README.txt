EN:
 This is an object recognition program that can be used on a real-time, live camera, or on video files using the simplified User Interface implemented. This program uses the YOLOv5s algorithm, deep learning and convolutional neural networks (CNNs) in order to achieve the desired object extraction and recognition. In order to close the program, press 'q' on the keyboard.

RO:
 Acest program de recunoastere a obiectelor poate fi folosit in timp real pe o camera live, sau poate fi aplicat pe fisierele video dorite folosind interfata de utilizator simpla si usor de folosit. Acest program foloseste arhitectura YOLOv5s, deep learning si retele neuronale convolutionale pentru a atinge detectarea si extragerea de obiecte dorita. Pentru a inchide programul, apasati 'q' pe tastatura.
